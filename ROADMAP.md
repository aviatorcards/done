# 🚀 Done. Roadmap

Tracking the journey from a minimal MVP to a powerful, collaborative Kanban experience.

---

## ✅ Phase 1: Foundation (Completed)
- [x] Initial architecture with Vapor 4 and Swift 6
- [x] Responsive layout with Tailwind CSS & Glassmorphism
- [x] Basic Kanban boards, columns, and cards
- [x] HTMX integration for near-instant interactions
- [x] Secure JWT-based authentication
- [x] Dark Mode support (System/Local preference)

## ✅ Phase 2: Collaboration & Ownership (Completed)
- [x] Multi-user board sharing (Many-to-Many relationships)
- [x] Invite Code system for new and existing users
- [x] Email Service integration (notifications & contact form)
- [x] Unified User Settings (Profile, Appearance, Privacy)
- [x] Data Export (User data portability in JSON format)
- [x] Permanent Account Deletion (GDPR/Data Privacy)
- [x] Refined Static Pages (About, Contact, Docs, Privacy)

## 🚧 Phase 3: Card Interactivity (In Progress)
- [ ] Card detail view with full description support
- [ ] Labels/Tags management (colors and names)
- [ ] Priority levels (Low, Medium, High, Urgent)
- [ ] Due dates with visual indicators
- [ ] Checklists within cards
- [ ] Column reordering via Drag & Drop (Native sortable integration)

## 🔮 Phase 4: Power User Features (Future)
- [ ] Board-wide Activity Feed ("Who did what")
- [ ] Global Search and Card Filtering
- [ ] Dashboard Widgets (Recent cards, due soon)
- [ ] Keyboard Shortcuts (Quick capture, navigation)
- [ ] PWA Support (Offline view & home screen icon)

## 📱 Phase 5: Mobile Experience
- [ ] Conceptual iOS App design
- [ ] Dedicated API endpoints for mobile clients
- [ ] Native iOS Application (SwiftUI)

---

> [!NOTE]  
> This roadmap is dynamic and based on community feedback and project priorities.
